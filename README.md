# JebacyrilArockiaraj.github.io

Personal academic website for GitHub Pages.

## Edit the website

- Update `index.html` with your current bio, links, news, publications, and projects.
- Replace `assets/profile.svg` with your own photo named `profile.jpg`, then update the image path in `index.html`.
- Add your CV as `assets/Jebacyril_Arockiaraj_CV.pdf`, or change the CV link.

## Publish

Commit and push to the `main` branch:

```sh
git add index.html styles.css assets README.md
git commit -m "Create academic homepage"
git push origin main
```

GitHub Pages will serve the site at:

```text
https://JebacyrilArockiaraj.github.io/
```
